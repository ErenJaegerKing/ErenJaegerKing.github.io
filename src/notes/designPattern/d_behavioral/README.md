**行为型模式用于描述程序在运行时复杂的流程控制，即描述多个类或对象之间怎样相互协作共同完成单个对象都无法单独完成的任务，它涉及算法与对象间职责的分配。**

行为型模式分为**类行为模式**和**对象行为模式**，前者采用**继承机制**来在类间分派行为，后者采用**组合或聚合**在对象间分配行为。由于**组合关系或聚合关系比继承关系耦合度低**，**满足“合成复用原则”**，所以对象行为模式比类行为模式具有更大的灵活性。
